 AI 聊天助手 (ai-chat-assistant)

  项目简介

  一个功能丰富的全栈 AI 智能聊天助手应用，基于智谱 AI 大模型，支持多轮对话、知识库 RAG、MCP 联网搜索、多角色切换、图片输入和文档上传等功能。

 GitHub: https://github.com/Aalen77/ai-chat-assistant

---
  核心功能

  1. AI 多轮对话

  - 基于智谱 AI GLM-4V-Flash 多模态模型
  - 流式逐字输出（SSE），打字机效果
  - 自动记住历史对话上下文，支持多轮追问
  - 历史超长时自动截断，防止超过模型 Token 上限

  2. 多角色 AI

  内置 6 种 AI 角色，每种都有独立的 System Prompt：

  ┌────────────┬────────────────────────────────┐
  │    角色    │              说明              │
  ├────────────┼────────────────────────────────┤
  │ 通用助手   │ 日常问答                       │
  ├────────────┼────────────────────────────────┤
  │ 编程助手   │ 代码编写、审查、调试、架构设计 │
  ├────────────┼────────────────────────────────┤
  │ 写作助手   │ 文章写作、润色、翻译、创意文案 │
  ├────────────┼────────────────────────────────┤
  │ 心理咨询师 │ 情感支持、心理建议             │
  ├────────────┼────────────────────────────────┤
  │ 学习导师   │ 知识讲解、引导式教学           │
  ├────────────┼────────────────────────────────┤
  │ 翻译助手   │ 多语种专业翻译                 │
  └────────────┴────────────────────────────────┘

  3. 知识库 RAG（检索增强生成）

  - 支持上传 PDF / Word / TXT / Markdown / CSV / JSON / 图片等多种格式文档
  - 支持拖拽上传文件夹，自动递归遍历子目录并保留目录树结构
  - 文本分块 + Embedding 向量化：调用智谱 Embedding-3 模型，按 500 字分块（50 字重叠）
  - 余弦相似度检索：用户提问时自动匹配知识库中最相关的 Top-3 文段
  - OCR 识别：图片文件通过 Tesseract.js 自动识别文字
  - 管理员权限控制（文档管理仅管理员可用）

  4. MCP 联网搜索

  - 集成 FireCrawl MCP（Model Context Protocol）服务
  - AI 可自主调用搜索、网页抓取等工具
  - 最多 5 轮工具调用循环，获取实时网络信息

  5. 图片输入

  - 支持上传图片（PNG/JPG/BMP/WebP）
  - 以多模态格式发送给 GLM-4V 模型进行视觉理解

  6. 用户认证系统

  - 注册 / 登录（BCrypt 密码加密 + JWT Token）
  - 首个注册用户自动成为管理员
  - 7 天 Token 有效期
  - 管理员专属的文档管理权限

  7. 会话管理

  - 多会话创建、切换、重命名、删除
  - 对话历史持久化存储（MySQL）
  - 清空当前会话消息
  - 自动用首条消息内容命名新会话

  8. 对话导出

  支持 3 种导出格式：
  - JSON — 结构化数据，含角色和消息
  - Markdown — 带标题格式化文本
  - 纯文本 — 简洁可读文本

  9. 思考过程可视化

  - 解析智谱 AI 返回的 <think>...</think> 标签
  - 以折叠面板展示模型推理过程（代码解析、数学推导等）
  - 支持流式输出中思考标签未闭合的中间状态

  10. UI / UX

  - 微信风格聊天界面（用户消息右、AI 消息左）
  - 暗色 / 亮色主题切换
  - 代码块语法高亮（Prism.js，支持多语言）
  - Markdown 富文本渲染
  - 一键复制 AI 回复
  - 响应式设计
